## Looker Janitor
Clean your Looker LookML view files.

## Getting Started

As a local script:
```shell
python main.py example.view.lkml
```

As a GitHub action:
```shell
- name: Run Looker Janitor
  uses: alhankeser/looker-janitor-action@v0
```

## Testing

Run the following command:
```
pytest
```
